## 第一章 需求分析

### 1.1 项目背景与目标

安多藏区是中国三大藏区之一，横跨青海、甘肃甘南、四川阿坝，拥有深厚的藏传佛教文化、游牧文化和民俗传统。然而，当前互联网上缺乏体系化、高质量的安多藏族文化展示平台，信息分散、内容碎片化，不利于文化的传播与保护。

本平台旨在构建一个**综合性安多藏族文化传播Web平台**，系统展示安多藏区的自然风光、历史人文、宗教文化与民俗传统，同时提供实用的藏文学习工具，推动安多藏族文化的数字化保护与传播。

### 1.2 面向用户

- 对藏族文化感兴趣的普通公众
- 计划前往安多藏区旅游的旅行者
- 藏语学习者和研究人员
- 文化保护与传承工作者

### 1.3 主要功能

| 功能模块 | 说明 |
|---------|------|
| 藏地旅游展示 | 覆盖青海、甘南、四川阿坝三大区域共25+景点专题页，含自然风貌、历史人文、文化意义、旅游服务 |
| 安多民俗介绍 | 展示安多藏族传统服饰、饮食、节庆、游牧生活等民俗文化 |
| 藏汉互译工具 | 基于MyMemory API的藏汉双向即时翻译，支持Ctrl+Enter快捷翻译 |
| 藏文字符查询 | 藏文字母表、元辅音查询与学习 |
| 藏语常用短语 | 日常交际、旅行用语等分类短语学习 |
| 藏历查询 | 集成藏历万年历（1991-2050），提供藏历与公历对照 |
| 民宿推荐 | 安多藏区特色民宿展示与详情浏览 |
| 景点详情 | 根据URL参数动态加载景点详细信息 |

### 1.4 竞品分析

| 维度 | 本平台 | 携程/马蜂窝藏区频道 | 藏文学习类APP |
|------|-------|-------------------|-------------|
| 文化深度 | ★★★★★ 系统化展示安多文化体系 | ★★★ 侧重旅游攻略 | ★★ 仅语言学习 |
| 区域覆盖 | ★★★★★ 青海+甘南+阿坝全覆盖 | ★★★★ 热门景点为主 | ★ 不涉及 |
| 藏文工具 | ★★★★ 翻译+字符+短语+藏历 | ★ 无 | ★★★★ 专业但单一 |
| 视觉设计 | ★★★★★ 统一红金藏式风格 | ★★★ 通用商业风格 | ★★★ 各异 |

### 3.2 关键技术

#### 3.2.1 Vue 3 响应式框架

首页和部分页面采用 Vue 3 `createApp` 进行状态管理：

```javascript
const { createApp } = Vue;
createApp({
  data() {
    return {
      showBackToTop: false,
      isLoggedIn: false,
      // ...
    }
  },
  methods: {
    handleScroll() {
      this.showBackToTop = window.pageYOffset > 300;
    },
    handlePlaceClick(place) {
      if (place.link) window.location.href = place.link;
    }
  }
}).mount('#app');
```

#### 3.2.2 藏汉互译 — MyMemory API 集成

翻译功能采用 MyMemory Translation API，实现免费、无需后端代理的浏览器端即时翻译：

```javascript
async function doTranslate() {
  const langpair = srcLang === 'zh' ? 'zh-CN|bo' : 'bo|zh-CN';
  const url = `https://api.mymemory.translated.net/get?q=${encodeURIComponent(q)}&langpair=${langpair}`;
  const resp = await fetch(url, { signal: AbortSignal.timeout(10000) });
  const data = await resp.json();
  if (data.responseStatus === 200) {
    outputText.value = data.responseData.translatedText;
  }
}
```

#### 3.2.3 卡片全区域点击交互

采用 CSS 绝对定位透明覆盖层，实现卡片任意位置均可点击跳转：

```css
.card { position: relative; }
.card-link {
  position: absolute;
  inset: 0;
  z-index: 1;
  cursor: pointer;
}
```

#### 3.2.4 响应式设计

所有页面均采用媒体查询实现移动端适配：

```css
@media (max-width: 768px) {
  .card-grid { grid-template-columns: 1fr; }
  .banner h1 { font-size: 36px; }
  .header-wrap { flex-direction: column; }
}
@media (max-width: 480px) {
  .banner { height: 300px; }
  .banner-stats { flex-direction: column; }
}
```

## 第五章 安装及使用

### 5.1 环境要求

- 任何现代Web浏览器（Chrome 90+, Firefox 90+, Safari 15+, Edge 90+）
- 网络连接（藏汉翻译功能需要联网）

### 5.2 安装与部署

#### 方式一：本地运行

```bash
# 1. 克隆代码仓库
git clone https://github.com/djsz123456/Qinghai-Amdo-Tibetan-Cultural-Communication-Platform.git

# 2. 进入项目目录
cd Qinghai-Amdo-Tibetan-Cultural-Communication-Platform

# 3. 使用任意HTTP服务器启动（以下任选其一）
# Python:
python -m http.server 8080
# Node.js:
npx serve .
# VS Code Live Server 插件：右键 index.html → Open with Live Server

# 4. 浏览器访问
http://localhost:8080
```

#### 方式二：在线访问

通过 GitHub Pages 直接访问：
`https://djsz123456.github.io/Qinghai-Amdo-Tibetan-Cultural-Communication-Platform/`

### 5.3 使用流程

1. **首页浏览**：打开平台首页，浏览"安多民俗"和"藏地旅游"两大板块
2. **景点探索**：点击感兴趣的景点卡片，进入景点专题页深入了解
3. **藏文工具**：通过首页"藏文实用工具"区域进入翻译、字符查询等功能
4. **藏汉翻译**：选择翻译方向 → 输入文字 → 点击"开始翻译"或按 Ctrl+Enter
5. **返回导航**：通过页面顶部导航栏随时返回首页或切换模块

## 第六章 项目总结

### 6.1 项目成果

本项目成功构建了一个涵盖25+景点专题页面的青海安多藏族文化传播平台，实现了以下核心目标：

- **文化体系化展示**：系统性地呈现了青海安多、甘南安多、四川阿坝安多三大区域的文化风貌
- **统一视觉语言**：所有页面采用一致的红金藏式设计风格，提供沉浸式的文化体验
- **实用工具集成**：提供藏汉互译、藏文字符查询、常用短语学习、藏历查询等实用功能
- **良好的用户体验**：全站支持响应式布局，卡片式交互设计，移动端友好

### 6.2 克服的困难

1. **藏汉翻译接口选型**：经历了从百度翻译API（需后端签名）→ 腾讯云TMT（存在CORS限制）→ MyMemory API（免费、免后端）的技术选型过程，最终选择了对浏览器端最友好的方案
2. **多区域内容编撰**：25+景点的文化内容需要兼顾准确性、可读性和文化敏感性，需要大量资料调研
3. **跨浏览器兼容**：藏文字符的渲染在不同浏览器中表现不一，通过字体回退策略解决

### 6.3 后续规划

- 增加用户互动社区功能（文化故事分享、旅行日记）
- 引入更专业的藏语翻译引擎，提升翻译质量
- 增加语音朗读功能，帮助用户学习藏语发音
- 扩展更多安多藏区景点和文化内容
- 开发移动端原生App版本
